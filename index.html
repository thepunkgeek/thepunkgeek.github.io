password-hasher

Below is the input for the open-source javascript password hasher that I've written. How does it work? It takes three inputs: A base password, the URL you want a password for, and the number of iterations to hash. The first thing it does create a SHA1 hash of the Password and URL individually, then it takes those hashes puts them together, and does sha256 hashes of that concatenation for the number of iterations specified.
