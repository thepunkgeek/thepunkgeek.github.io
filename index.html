password-hasher

Above is a front-end for the open-source javascript password hasher that I've written.

It works by taking three inputs:
A base password, the URL you want a password for, and the number of iterations to hash. The first thing it does is create a SHA1 hash of the Password and URL individually, then it takes those hashes and puts them together, and does SHA256 hashes of that concatenation for the number of iterations specified.
